# Guías Interactivas

Esta carpeta contiene las guías interactivas del SENA.

## Organización de las Guías

Cada guía debe incluir:
- Un título descriptivo
- Objetivos de aprendizaje
- Contenido interactivo
- Actividades de evaluación

## Formato de URLs

Las URLs de las guías interactivas siguen este formato:

```
https://github.com/MarcelaLoaiza/GU-AS-INTERACTIVAS/blob/main/guias/[nombre-de-la-guia]/
```

O si las guías están hospedadas externamente:

```
https://[plataforma-externa]/guias/[nombre-de-la-guia]
```
